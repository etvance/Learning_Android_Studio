# Hangman Mobile App

This project is a simple Android Hangman game built using Kotlin in Android Studio. The app allows the user to guess letters to reveal a hidden word while tracking remaining guesses. The display updates after each guess, including both the word progress and a hangman image. This project demonstrates basic mobile app development concepts such as user input, UI updates, images, and app structure.

---

## Instructions for Build and Use

### Steps to build and/or run the software:

1. Install Android Studio
2. Open the project folder in Android Studio
3. Allow Gradle to sync and install any required dependencies
4. Run the app using an Android emulator or a physical Android device

---

### Instructions for using the software:

1. Launch the app
2. Enter a letter into the input field
3. Press the "Guess" button
4. The word display and hangman image will update based on your guess
5. Continue guessing until you win or lose

---

## Development Environment

To recreate the development environment, you need the following software and/or libraries:

* Android Studio (latest version recommended)
* Kotlin (included with Android Studio)
* Java Development Kit (JDK 17 or newer)
* Android Emulator or Android device
